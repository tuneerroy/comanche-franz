Broker Leader:
/{producer id}/topics - POST
- adds topic

/{producer id}/topics/{topic} - DELETE
- deletes topic

/groups/{group_id} - POST
- adds consumer to consumer group with given id

/groups/{group_id} - DELETE
- removes consumer from consumer group with given id

/groups/{group_id}/topics/{topic} - POST
- adds topic to consumer group with given id

/groups/{group_id}/topics/{topic} - DELETE
- removes topic from consumer group with given id

Broker:
/{topic}/messages - POST
- writes message to file for topic

/{topic}/messages?offset="" - GET
- reads from file to get most recent message

Consumer:
/partitions - PUT
- update topic/partition distribution
